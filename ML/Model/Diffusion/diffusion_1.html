Preliminaries

对数据的认识

机器学习就是对一个未知分布的数据建模的过程。无论是机器学习哪种学派，其都认为观察到的数据并不是凭空产生的，而是由一个潜在的、客观存在的数据生成过程所产生。这个数据生成过程可以用一个概率分布来描述。

例如抛硬币，会出现正面或反面，我们抛了kkk次，得到kkk个数据。这个结果就可以看作是由一个伯努利分布生成（采样）的。

因此，我们可以得出以下总结：

假设观测数据x1,x2,...,xnx_1, x_2, ..., x_nx1​,x2​,...,xn​是从某个真实但未知的分布Pdata(x)P_{data}(x)Pdata​(x)中独立采样得到的。为了表示这个数据生成过程，我们可以使用一个参数化的模型Pmodel(x∣θ)P_{model}(x|\theta)Pmodel​(x∣θ)来近似真实分布，这里的θ\thetaθ就是对该参数化模型的描述，一个好的θ\thetaθ会让我们的参数化模型更好地接近真实的数据分布，因此我们的目标就是优化参数 θ\thetaθ使模型分布尽可能接近真实数据分布。

如何优化参数θ\thetaθ呢，这就引出了机器学习的两大学派——频率学派以及贝叶斯学派。

频率学派

频率派认为上面的Pmodel(x∣θ)P_{model}(x|\theta)Pmodel​(x∣θ)中的θ\thetaθ是一个未知的但客观存在的常量，可以通过现有数据进行直接的估计。

回到上面抛硬币例子，假如kkk次实验中，有aaa次向上的结果，频率派认为，这不是偶然的，这个已发生的事件肯定反映了硬币向上的真实概率。所以我们要做的就是让出现aaa次朝上，k−ak-ak−a次朝下这种事件的概率最大，把模型中的参数θ\thetaθ依据数据给估计出来，这样就可以还原真实世界对于抛硬币的情况了（即以后抛硬币都像这次观测的一样）。

实际上，Pmodel(x∣θ)P_{model}(x|\theta)Pmodel​(x∣θ)对应的就是似然函数的概念，即对于某一θ\thetaθ，出现当前数据x1,x2,...,xnx_1, x_2, ..., x_nx1​,x2​,...,xn​的概率大小，上述过程可以表示为：

L(θ)=L(x1,x2,x3,…,xn;θ)=∏i=1np(xi;θ) L(\theta)=L(x_1,x_2,x_3,\ldots,x_n;\theta)=\prod_{i=1}^np(x_i;\theta) L(θ)=L(x1​,x2​,x3​,…,xn​;θ)=i=1∏n​p(xi​;θ)

也可以写成以下式子：

P(x∣θ)=∏i=1np(xi;θ) P(x|\theta) = \prod_{i=1}^np(x_i;\theta) P(x∣θ)=i=1∏n​p(xi​;θ)

这就是似然函数。似然函数反应了，假定参数θ\thetaθ下，取得当前观测结果的概率大小（也就是当前的观测结果），那么这个概率越大，就越说明参数θ\thetaθ的准确，也就是说模型逼近真实分布的程度越好。

所以频率学派就是通过优化参数θ\thetaθ使得似然函数尽可能的大，从而使出现当前这种情况的概率最大。那么如何最大化似然函数呢？

这就引入了我们熟悉的最大似然估计 (MLE):

θMLE=arg⁡max⁡θ∑i=1Nlog⁡p(xi∣θ) \theta_{MLE}=\underset{\theta}{\arg \max } \sum_{i=1}^{N}\log p\left(x_{i} \mid \theta\right) θMLE​=θargmax​i=1∑N​logp(xi​∣θ)

即把问题转换为了一个优化问题，其遵循的优化过程为：

  1. 建立模型，该模型可以是一个简单的线性回归 (那么斜率kkk和偏置bbb就是我们要估计的参数)，或是一个神经网络 (很难显式表达，网络里所有权重就是我们要估计的参数)；
  2. 设计损失函数，如上述例如常见的MSE (回归问题)，或是交叉熵 (分类问题)；
  3. 优化，例如梯度下降，牛顿法等。

贝叶斯学派

贝叶斯学派对参数θ\thetaθ的看法与频率派完全不同。贝叶斯派认为θ\thetaθ不是一个未知的固定常量，而是一个随机变量。这意味着θ\thetaθ本身是不确定的，它有自己的概率分布，这个概率分布就是先验。

因此，贝叶斯学派的核心思想不是去估计一个唯一的、真实的θ\thetaθ值，而是要根据观测到的数据+先验知识，来更新我们对θ\thetaθ不确定性的认知。

回到抛硬币的例子，贝叶斯派在抛硬币之前，就会对硬币朝上的概率θ\thetaθ有一个先验（信念），用一个概率分布P(θ)P(\theta)P(θ)来表示，例如：

  1. 如果我们认为这很可能是一枚均匀的硬币，我们可能会用一个在0.5处有高概率、两端概率低的Beta分布来作为先验。
  2. 如果我们对硬币一无所知，我们可能会使用均匀分布U(0,1)U(0, 1)U(0,1)作为先验，即认为θ\thetaθ取[0, 1]中任意值的可能性都一样（即无信息先验）。

然后，我们抛kkk次，出现aaa次朝上的观测数据，这可以用前面提到的似然函数P(x∣θ)P( x| \theta)P(x∣θ)来表示。

所以，我们可以用P(x∣θ)P(θ)P(x | \theta) P(\theta)P(x∣θ)P(θ)来表示发生当前观测数据的情况。

由于θ\thetaθ可能有许多不同的取值，对应每种取值都有可能发生当前观测数据xxx的情况，那么对应所有不同θ\thetaθ下，发生观测xxx的情况可以表示为：p(x)=∫P(x∣θ)P(θ)dθp(x) = \int P(x | \theta) P(\theta) d\thetap(x)=∫P(x∣θ)P(θ)dθ (或 p(x)=∫P(x,θ)dθp(x) = \int P(x, \theta)d\thetap(x)=∫P(x,θ)dθ)。

因此，发生当前观测，对应某一θ\thetaθ的概率大小就可以表示为：

P(x∣θ)P(θ)P(x)=P(x∣θ)P(θ)∫P(x∣θ)P(θ)dθ \frac{P(x | \theta) P(\theta)}{P(x)}=\frac{P(x | \theta) P(\theta)}{\int P(x | \theta) P(\theta) d\theta} P(x)P(x∣θ)P(θ)​=∫P(x∣θ)P(θ)dθP(x∣θ)P(θ)​

这其实就是当前情况的一种归一化。

我们把这种对于当前情况的描述称为后验概率，即:

P(θ∣x)=P(x∣θ)P(θ)P(x)=P(x∣θ)P(θ)∫P(x∣θ)P(θ)dθ P(\theta | x) = \frac{P(x | \theta) P(\theta)}{P(x)}=\frac{P(x | \theta) P(\theta)}{\int P(x | \theta) P(\theta) d\theta} P(θ∣x)=P(x)P(x∣θ)P(θ)​=∫P(x∣θ)P(θ)dθP(x∣θ)P(θ)​

这其实也就是我们常见的贝叶斯公式。那么贝叶斯派就是要真正掌握这样一个后验分布，也就是说，给出当前观测，对应的θ\thetaθ不同取值的概率大小，或者说有多大概率是当前θ\thetaθ产生的当前观测。

OK，这里我们暂停一下，回去与频率派对比下，你会发现，频率派就是在最大化上面的似然，因为频率派认为θ\thetaθ是一个固定值，他不存在其他情况，所以可以直接用似然表示，但贝叶斯派认为，θ\thetaθ可能有很多取值，都有可能导致现在情况发生，因此写出了这样一种归一化的形式，要找到一种普世的，“”掌控全局“”的表达。

总结一下上面的表达式各项含义：

  • P(θ∣D)P(\theta | D)P(θ∣D)是后验概率：在观测到数据xxx之后，我们对θ\thetaθ的信念分布。这是贝叶斯推断的核心目标。
  • P(D∣θ)P(D | \theta)P(D∣θ) 是似然：这与频率派中的似然函数L(θ)L(\theta)L(θ)是同一个东西，就是当前的观测结果。
  • P(θ)P(\theta)P(θ)是先验：在观测数据xxx之前，我们对θ\thetaθ的初始信念。
  • P(x)P(x)P(x) 是一个归一化常数，也叫做证据或边际似然，它确保后验概率分布P(θ∣x)P(\theta | x)P(θ∣x)对所有 θ\thetaθ积分后等于1。

贝叶斯学派的重心在于求解完整的后验分布P(θ∣x)P(\theta | x)P(θ∣x)。这个分布包含了给定数据和先验信念下，关于θ\thetaθ的所有信息。但在实际应用中，计算P(x)P(x)P(x)的积分（尤其是在θ\thetaθ高维时，如神经网络）通常是极其困难的。 因此，人们提出了一种简化的方法，即只寻找后验分布P(θ∣x)P(\theta | x)P(θ∣x)中概率密度最大的那个点 θ\thetaθ，以此作为θ\thetaθ的一个代表性估计。这就是最大后验估计 (MAP)：

θMAP=arg⁡max⁡θP(θ∣x) \theta_{MAP} = \underset{\theta}{\arg \max } P(\theta | x) θMAP​=θargmax​P(θ∣x)

由于P(x)P(x)P(x)是一个与θ\thetaθ无关的常数，在优化时可以被忽略：

θMAP=arg⁡max⁡θP(x∣θ)P(θ)P(x)=arg⁡max⁡θP(x∣θ)P(θ) \theta_{MAP} = \underset{\theta}{\arg \max } \frac{P(x | \theta) P(\theta)}{P(x)} = \underset{\theta}{\arg \max } P(x | \theta) P(\theta) θMAP​=θargmax​P(x)P(x∣θ)P(θ)​=θargmax​P(x∣θ)P(θ)

为了计算方便，我们同样可以对其取对数

θMAP=arg⁡max⁡θ(log⁡P(x∣θ)+log⁡P(θ)) \theta_{MAP} = \underset{\theta}{\arg \max } \left( \log P(x| \theta) + \log P(\theta) \right) θMAP​=θargmax​(logP(x∣θ)+logP(θ))

如果我们假设数据是独立同分布的，那么似然P(x∣θ)P(x | \theta)P(x∣θ)就可以像MLE一样被展开：

θMAP=arg⁡max⁡θ(∑i=1Nlog⁡p(xi∣θ)+log⁡P(θ)) \theta_{MAP} = \underset{\theta}{\arg \max } \left( \sum_{i=1}^{N} \log p(x_i | \theta) + \log P(\theta) \right) θMAP​=θargmax​(i=1∑N​logp(xi​∣θ)+logP(θ))

通过比较θMAP\theta_{MAP}θMAP​和θMLE\theta_{MLE}θMLE​的公式： 我们可以得出一个非常非常重要的结论： 最大后验估计 (MAP) = 最大似然估计 (MLE) + 先验信念 log⁡P(θ)\log P(\theta)logP(θ)

MAP的优化过程如下：

  1. 建立模型。
  2. 设计损失函数。在MAP中，损失函数不仅仅是似然（如MSE或交叉熵），还额外包含了一个先验项log⁡P(θ)\log P(\theta)logP(θ)。
  3. 优化（同MLE）。

设计损失函数。在MAP中，损失函数不仅仅是似然（如MSE或交叉熵），还额外包含了一个先验项log⁡P(θ)\log P(\theta)logP(θ)。这个log⁡P(θ)\log P(\theta)logP(θ)项通常充当了正则化项 (Regularizer)的角色，它会惩罚那些与先验信念偏差太大的θ\thetaθ值。那么常见的正则化方式就是我们所说的L1/L2正则化。

**从MAP视角解读正则化技术：**

以L2正则化为例，它在损失函数中增加了一项

λ∣∣θ∣∣2 \lambda ||\theta||^2 λ∣∣θ∣∣2

这里θ\thetaθ就是神经网络的参数，过去我们直观理解为什么L2正则化可以防止过拟合，因为它使得网络参数趋向于0，类似于dropout随机抑制某些神经元的输出，从而减小模型复杂度，从而减小过拟合风险。

实际上，这里的L2正则化表达等价于假设参数θ\thetaθ的先验分布P(θ)P(\theta)P(θ)是一个均值为一个均值为0、方差为σp2\sigma_p^2σp2​的高斯分布。

这个先验P(θ)P(\theta)P(θ)的概率密度函数可以写作：

P(θ)=∏i=1d12πσp2exp⁡(−θi22σp2)=C⋅exp⁡(−∣∣θ∣∣22σp2) P(\theta) = \prod_{i=1}^{d} \frac{1}{\sqrt{2\pi\sigma_p^2}} \exp\left(-\frac{\theta_i^2}{2\sigma_p^2}\right) = C \cdot \exp\left(-\frac{||\theta||^2}{2\sigma_p^2}\right) P(θ)=i=1∏d​2πσp2​​1​exp(−2σp2​θi2​​)=C⋅exp(−2σp2​∣∣θ∣∣2​)

（其中 ddd 是参数 θ\thetaθ 的维度，CCC 是一个不依赖于 θ\thetaθ 的归一化常数）

根据我们在贝叶斯学派笔记中推导的最大后验估计 (MAP) 公式，我们优化的目标是：

θMAP=arg⁡max⁡θ(log⁡P(x∣θ)+log⁡P(θ)) \theta_{MAP} = \underset{\theta}{\arg \max } \left( \log P(x | \theta) + \log P(\theta) \right) θMAP​=θargmax​(logP(x∣θ)+logP(θ))

现在，我们把高斯先验的对数 log⁡P(θ)\log P(\theta)logP(θ) 计算出来：

log⁡P(θ)=log⁡(C⋅exp⁡(−∣∣θ∣∣22σp2))=log⁡C−12σp2∣∣θ∣∣2 \log P(\theta) = \log \left( C \cdot \exp\left(-\frac{||\theta||^2}{2\sigma_p^2}\right) \right) = \log C - \frac{1}{2\sigma_p^2}||\theta||^2 logP(θ)=log(C⋅exp(−2σp2​∣∣θ∣∣2​))=logC−2σp2​1​∣∣θ∣∣2

将它代回MAP的优化目标中。由于 log⁡C\log ClogC 是一个常数，它不影响最优化时 θ\thetaθ 的取值，所以可以被忽略：

θMAP=arg⁡max⁡θ(log⁡P(x∣θ)−12σp2∣∣θ∣∣2) \theta_{MAP} = \underset{\theta}{\arg \max } \left( \log P(x | \theta) - \frac{1}{2\sigma_p^2}||\theta||^2 \right) θMAP​=θargmax​(logP(x∣θ)−2σp2​1​∣∣θ∣∣2)

上述表达中，最大化MAP就是最小化 12σp2∣∣θ∣∣2\frac{1}{2\sigma_p^2} ||\theta||^22σp2​1​∣∣θ∣∣2，对应了机器学习中最小化损失函数。

即：

θMAP=arg⁡min⁡θ(Loss+12σp2∣∣θ∣∣2) \theta_{MAP} = \underset{\theta}{\arg \min } \left( \text{Loss} + \frac{1}{2\sigma_p^2}||\theta||^2 \right) θMAP​=θargmin​(Loss+2σp2​1​∣∣θ∣∣2)

定义 λ=12σp2\lambda = \frac{1}{2\sigma_p^2}λ=2σp2​1​，这个公式就变成了我们熟悉的L2正则化的形式：

θMAP=arg⁡min⁡θ(Loss+λ∣∣θ∣∣2) \theta_{MAP} = \underset{\theta}{\arg \min } \left( \text{Loss} + \lambda ||\theta||^2 \right) θMAP​=θargmin​(Loss+λ∣∣θ∣∣2)

所以，L2正则化在数学上就等价于假设参数θ\thetaθ满足均值为0的高斯先验的MAP估计，先当于向网络注入了一定先验知识，帮助网络向某个方向更新收敛。

贝叶斯预测

MAP只是一种“点估计”，它只找到了后验分布 P(θ∣x)P(\theta | x)P(θ∣x) 中概率最大的那个点 θMAP\theta_{MAP}θMAP​。这在实践中很有用（例如L2正则化），但它并没有得到完整的后验分布。

为什么我们如此需要这个完整的分布呢？因为它可以用来进行贝叶斯预测 (Bayesian Prediction)。

假设我们有了一个新的、未见过的数据点 x~\tilde{x}x~，我们想预测它的概率。我们不是用某一个 θMAP\theta_{MAP}θMAP​ 来进行预测 P(x~∣θMAP)P(\tilde{x} | \theta_{MAP})P(x~∣θMAP​)，而是利用整个后验分布，计算一个加权平均。这个权重就是 θ\thetaθ 的后验概率 P(θ∣x)P(\theta | x)P(θ∣x)。

这就是后验预测分布 (Posterior Predictive Distribution)，其公式如下：

P(x~∣x)=∫P(x~∣θ)P(θ∣x)dθ P(\tilde{x} | x) = \int P(\tilde{x} | \theta) P(\theta | x) d\theta P(x~∣x)=∫P(x~∣θ)P(θ∣x)dθ

其含义是：我们考虑了所有可能的参数 θ\thetaθ，并根据我们观察到的数据 xxx 来判断它们各自的可能性 P(θ∣x)P(\theta | x)P(θ∣x)，然后将它们对 x~\tilde{x}x~ 的预测 P(x~∣θ)P(\tilde{x} | \theta)P(x~∣θ) 进行积分，本质就是加权求和。

这样做能完美地量化不确定性（考虑所有可能的情况）。
